# Enterprise (planned)

This directory is a placeholder for the future enterprise tier: hosted dashboard, Entra ID SSO for the tool itself, org-wide policy management, audit trail, and compliance framework mapping (SOC2/PCI).

Nothing here is built yet — this file exists to reserve the module boundary so the open-source core (everything outside this directory) and any future paid features stay cleanly separated from day one, per the licensing model in the root [README](../README.md).

See [`DESIGN.md`](DESIGN.md) for the current scope/architecture sketch and the open product decisions blocking implementation.

Community support for the OSS core happens via GitHub issues. Enterprise support (SLA-backed) is planned as a separate offering once this module exists.
